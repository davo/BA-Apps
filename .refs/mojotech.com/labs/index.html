Mojo Labs.

We love to make the tools and communities we interact with more enjoyable.

allocate

Simple resource planning for agencies, studios, and firms. We built Allocate to scratch our own itch. We were tired of using spreadsheets to manage our project assignments and had a hunch other agencies were too. Allocate is actually fun to use, and updates in real-time so any changes you make are broadcast to your entire team. Allocate saves us (and hopefully others) hours every week and helps us answer our favorite question: When can you start?

Here are some open source projects we built to make our lives easier (YMMV).

stickymojo

stickyMojo is a contained sticky sidebar plugin for jQuery. It’s lightweight, fast, flexible, and compatible with Firefox, Chrome, Safari, and IE8+. It will degrade gracefully in older versions of IE.

hipchat

An HTTP API Wrapper in Ruby with Capistrano hooks.

vire.js

Vire.js is a jQuery plugin that extends the froogaloop library for the Vimeo player. It allows you to execute code at a given point in time in a video.

pie chart

PieChart is a simple way to make beautiful donut-style graphs. It has the options for angular gradient fills and animations and is available as a plugin for ease of implementation.

batchelor

Batchelor will override find_in_batches and find_each in ActiveRecord descendant classes that do not have an integer primary key as these will not work when using PostgreSQL.

pullr

pullr is a command line interface to create GitHub pull requests based on the state of your local Git repository

semaphore status

Semaphore Status is a command line interface to the Semaphore hosted continuous integration service. It takes the state of your local Git repository into account when displaying Semaphore’s feedback.

blimey brackets

Blimey Brackets is a bracket-closing detection for Sublime Text 2. It finds the matching opening curly bracket for the closing bracket left of the text cursor and displays the text preceding the opening bracket on the status line. A useful tool for knowing which CSS rule you’re editing.

nearest

Nearest is a Node.js utility for finding the nearest x minutes for a given time.
