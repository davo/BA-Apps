The code-savvy web design tool

Stop writing code. Start drawing it.

Macaw provides the same flexibility as your favorite image editor but also writes semantic HTML and remarkably succinct CSS. It's time to expect more from a web design tool.

Built for today's web designer.

Fluid canvas and grid

Macaw is engineered for fluid widths. Resize the canvas and your grids and elements will update accordingly.

Responsive Breakpoints

Designing for different device sizes is tricky. Macaw lets you define arbitrary breakpoints and tweak properties for that specific width.

Rich Typography

In addition to a full library of web fonts and rich-text editing, Macaw provides you with useful calculations like characters per line.

Interactivity

Building clickable prototypes in Macaw is as easy as clicking on a target, but it doesn't stop there. You can also add more dynamic interactions with the scripting palette. Macaw will even autogenerate templates and jQuery variables for you.

Design more with less.

Global Classes

Macaw lets you save common styles so that you can quickly apply them to elements across your document. Once applied, you can update all of the elements at once with the class editor.

reusable components and assets

Have a complex element that you need to use on multiple pages? With Macaw you can store it in your library as a component and drag it on to any other page in your project. Edit it in one spot and all other instances will be automatically updated.

Morphing

Wouldn't it be nice if you could convert a wireframe into a full-blown mockup? With Macaw you can. Macaw is built on a flexible, component-based architecture which allows elements to be morphed into different things. For example, you can morph a static wireframe video element into an actual YouTube video.

The engine that learns as you design.

We wanted Macaw to write code you'd be proud enough to hand to developers. Current applications rely on trendy frameworks, export rigid positioning and generate ridiculous IDs. We think designers and developers deserve better. Macaw's powerful, patent-pending engine is called Alchemy and it can do some incredible things.

Consolidation

Alchemy analyzes every possible selector to describe your document and determines the most succinct way of using them based on logical criteria: applicability, specificity and semantics.

Absolute to Static Conversion

Macaw offers designers the freedom of absolute positioning within the interface but will convert to a static document flow upon export by calculating the necessary margins, padding, floats and clears. It's the best of both worlds.

Style Guides

Designing for the web requires more than mockups. It requires thorough design systems that can be implemented in a modular fashion. Macaw helps you abstract the finer details of your project by automatically generating a style guide. Macaw style guides outline the typography, colors, grids, classes and other components used throughout your project so that you, your client and your developer are always on the same page.
